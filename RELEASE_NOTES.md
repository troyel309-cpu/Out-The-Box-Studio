# Out the Box Studio v1.0.0

## OpenAI Build Week submission

This release establishes the first public baseline of Out the Box Studio.

### Included

- Local Node.js studio runtime
- Creative project storage
- Specialized worker registry and crew assembly
- Capability-based routing
- Character continuity controls
- Reference image intake
- Hybrid, My Picture Only, and AI Only modes
- Video-studio planning
- Server-side OpenAI key handling
- GitHub Actions syntax verification

### Evidence status

Built and locally verified. Production-scale usage and measurable outcome proof remain future milestones.
