# OpenAI Build Week Submission

## Project

Out the Box Studio

## One-line description

A local-first AI creative production studio that turns an idea into organized work by assembling specialized workers, preserving references, and managing production assets.

## Problem

Creators often work across disconnected prompts, files, and AI tools. This makes continuity, organization, and repeatable production difficult.

## Solution

Out the Box Studio identifies required capabilities, assembles a specialized crew, builds an execution plan, and keeps characters, references, assets, and evidence together.

## Differentiators

- Specialized worker architecture
- Character continuity controls
- Owner-controlled reference assets
- Multiple production modes
- Local-first ownership
- Evidence-first status language
- Server-side secret handling
