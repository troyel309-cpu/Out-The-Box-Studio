# Demo Script

Hi, I’m Troy Campbell, and this is Out the Box Studio.

Most AI tools give you one assistant. Out the Box Studio is designed to give creators an organized production workforce.

A creative idea can quickly become scattered across prompts, files, characters, and tools. The creator ends up carrying every job.

Out the Box Studio starts with one idea. It identifies the capabilities required, assembles qualified workers, creates an execution plan, and keeps references and assets organized.

I can create a project, enter a production request, upload an official reference image, and choose Hybrid, My Picture Only, or AI Only mode.

Registered character rules help keep a character consistent instead of redesigning it every time.

The goal is not just to generate something once. The goal is to create a repeatable production system where the work moves through the studio.

This is the first verified baseline of Out the Box Studio, built for OpenAI Build Week.
